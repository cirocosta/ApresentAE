Teste básico para client de GCM

Testar Já!

Sobre

GCM Testing é um pequeno projeto para que as pessoas possam, antes de iniciar a implementação do server-side do GCM (Google Cloud Messaging), testar se o client-side foi devidamente configurado.

O serviço poderá ser corretamente utilizado quanto o status de Aparelho Conectado for Aguardando aparelhos. Se persistir a mensagem Abrindo canal atualize a página e tente de novo (um novo código será gerado) e deixe a janela aberta. O aparelho será reconhecido automaticante se a implementação for correta no client-side.

Código

Todo o código, tanto deste testador quanto de uma implementação client-side padrão estão disponíveis no GitHub.

O código da implementação de um client que efetiva a comunicação com este serviço está disponível aqui. Está também disponível o código da aplicação web.
